# Tool Notes

## Web Search
- Use SearXNG (via one-search MCP) for all web searches
- Do NOT use Brave search
- Preferred search categories: science, news, general
- Always search in English for better coverage of quantitative finance literature

## GitHub
- Use git CLI for all repo operations
- Always pull before reading, push after writing
- Respect write permissions: only write to strategies/candidates/, evo/, memory/, knowledge/

## Research Workflow
- For arXiv: search via SearXNG with site:arxiv.org filter
- For SSRN: search via SearXNG with site:ssrn.com filter
- Save structured paper summaries to knowledge/papers/
